ローンの種類と特徴

住宅ローン

住宅金融支援機構と民間金融機関が提携した全期間固定金利型の「フラット３５」、民間金融機関による「住宅ローン」があります。 住宅ローン金利には、「固定期間選択型」や「変動型」があります。

リフォームローン

自宅の土地や建物を担保にする「有担保型」と担保の必要がない「無担保型」があります。金利は一般住宅ローンより高めに設定されていることが多いです。

一体型ローン

中古住宅の代金とリフォーム費用をまとめて住宅ローン金利で借りられます。 融資実行を物件の残金決済時とリフォーム完成後の２回行うため、売り主さまにご負担を掛けることが少なくなります。
